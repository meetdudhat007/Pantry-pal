# Pantry Pal Frontend (React + Vite)

AI-powered recipe recommendation platform built with React and Tailwind CSS.

## Setup

```bash
cd frontend-react
npm install
npm run dev
```

Server runs on `http://localhost:3000`

## Backend Requirements

The frontend expects the backend API to be running on `http://localhost:4000`

Make sure the backend server is running:
```bash
cd backend
npm start
```

## Features

- 🤖 AI Chatbot with Gemini integration
- 🍲 Recipe generation and search
- 🏷️ Diet filtering (Veg, Vegan, Keto, Protein)
- 💾 MongoDB recipe storage
- 📱 Mobile-friendly responsive design

## Technologies

- React 18
- Vite (fast build tool)
- Tailwind CSS
- Fetch API for backend communication
